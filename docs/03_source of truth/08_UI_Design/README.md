# UI Design

## Deskripsi

Folder ini berisi rancangan antarmuka (UI) aplikasi SpeakUp.

## Struktur

- Login & Signup
  - Tampilan login dan registrasi.

- Siswa
  - Tampilan yang digunakan oleh pengguna siswa.

- BK
  - Tampilan yang digunakan oleh Guru BK.

- Kepsek
  - Tampilan yang digunakan oleh Kepala Sekolah.

## Format

Seluruh desain disimpan dalam format PNG sebagai hasil ekspor dari Figma.
