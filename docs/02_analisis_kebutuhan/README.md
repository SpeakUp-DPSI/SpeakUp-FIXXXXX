Analisis Kebutuhan Sistem

Folder ini berisi dokumen analisis kebutuhan untuk **Sistem Pelaporan dan Penanganan Kasus Perundungan di SMK 4 Muhammadiyah Yogyakarta**.

## Isi Folder

- `Analisis_Kebutuhan_Sistem.docx`  
  Memuat problem statement, stakeholder, kebutuhan fungsional, traceability table, dan validasi hasil analisis.

- `use_case_diagram.docx`  
  Memuat aktor, user requirement, use case diagram, dan penjelasan interaksi pengguna dengan sistem.

## Permasalahan Utama

- Belum tersedia pelaporan anonim.
- Validasi laporan masih dilakukan secara manual.
- Pencatatan dan tindak lanjut kasus belum terstruktur.
- Pemantauan perkembangan kasus belum dilakukan melalui satu sistem.

## Stakeholder

- Siswa
- Guru BK
- Kepala Sekolah
- Orang Tua/Wali

## Tujuan

Dokumen ini menjadi acuan dalam perancangan dan pengembangan sistem pelaporan perundungan yang aman, terstruktur, dan mudah dipantau.
